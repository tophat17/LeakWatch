# Screenshots

Drop your web-UI screenshots in this folder so they appear in the main README:

- `dashboard.png`  - the main page showing the container list, badges and statuses
- `checks.png`     - hovering a status to reveal the per-container scan checklist
- `learnmore.png`  - a container's "About this app" detail panel (click a name)

PNG or JPG are fine. Keep them reasonably sized (e.g. 1200-1600px wide).
